Elementos de texto

Formularios

form Define un formulario form

method="[GET||POST]" Establece el método usado para enviar la información

action="[PAGINA]" Establece el dónde se mandará la información

labelDefine una etiqueta para un input label

for="[ID]"Especifica el input con el que debe estar la etiqueta

form="[ID]"Especifica el form con el que debe estar la etiqueta

input Define un campo de introducción de datos input

type="Tipos" Establece el tipo de información a pedir (Text por defecto)

name="[NOMBRE]"Nombre que identifica el input

value="[VALOR_POR_DEFECTO]"Establece el valor por defecto del input

min="[NÚMERO]" max="[NÚMERO]" step="[NÚMERO]"Establece el mínimo, máximo y cuanto avanza de un input number

list="[#ID_DATALIST]"Establece inputs similares en base a un datalist

required Establece si es un input obligatorio

autofocus Establece el foco en el input cuando carga la página

disabled Establece el input en desactivado

placeholder=[VALOR] Establece un valor por defecto en el input

minlength/maxlength="[NÚMERO_DE_CARACTERES]"Establece el mínimo/máximo de letras de un input tipo text

pattern=[EXPRESION_REGULAR] Establece la validez de un input tipo text mediante una expresión regular

textareaDefine un área de textotextarea

name="[NOMBRE]"Nombre que identifica el textarea

rows="[#NÚMERO]" cols="[#NÚMERO]"Establece tamaño en filas y columnas del textarea

buttonDefine un botónbutton

type="button"Le dice al navegador de interpretar el button como un button

select Define un campo lista con diferentes opciones select
option Define una opción en la lista option

name="[NOMBRE]"Nombre que identifica el select

value="[VALOR]"Establece el valor por defecto del option

datalist Define una lista de recomendaciones de un input datalist
option Define una opción en la lista option

value="[VALOR]"Establece el valor del option
